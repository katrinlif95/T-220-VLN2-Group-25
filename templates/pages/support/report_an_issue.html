Report an Issue

If you experience technical issues while using ArtVault, we encourage you to report them so our team can investigate and improve the platform. This may include problems related to bidding, payments, account access, artwork listings or general site functionality.

When reporting an issue, please include as much relevant information as possible, such as the page where the problem occurred, any error messages displayed and the actions that led to the issue. Detailed reports help us resolve problems more efficiently.
